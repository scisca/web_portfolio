Tentang Saya

Saya Franscisca Sriwahyuni, seorang mahasiswa Program Studi Sistem Informasi di Universitas Katolik Soegijapranata. Selama kuliah, saya aktif berorganisasi dan terlibat dalam berbagai program kerja yang membekali saya dengan kemampuan memecahkan masalah, berkolaborasi, dan berkomunikasi dengan baik. Ketertarikan saya pada dunia teknologi dan perkembangan sistem informasi membuat saya selalu antusias untuk belajar hal-hal baru. Saya telah memperdalam pengetahuan dalam beberapa bahasa pemrograman seperti PHP, Java, JavaScript, CSS, HTML, dan SQL melalui kursus online dan proyek pribadi. Dalam karir saya, saya bercita-cita untuk berkontribusi dalam menciptakan solusi teknologi inovatif yang dapat membantu meningkatkan efisiensi dan efektivitas sistem informasi. Selain belajar, saya juga menikmati waktu untuk membaca buku, hiking, dan mencoba resep masakan baru.

3. Mini Project To-Do-List

To Do List ini adalah aplikasi daftar tugas sederhana yang membantu pengguna mengelola kegiatan sehari-hari mereka. Pengguna dapat menambahkan, mengedit, dan menghapus tugas serta menandai tugas yang telah selesai.

Link To-Do-List

4. Mini Project Weather atau Perkiraan Cuaca

Proyek ini adalah aplikasi cuaca yang memberikan informasi terkini tentang kondisi cuaca di lokasi tertentu. Pengguna dapat memasukkan nama kota dan aplikasi akan menampilkan suhu, kondisi cuaca, dan prakiraan cuaca beberapa hari ke depan. Aplikasi ini menggunakan API cuaca eksternal untuk mengambil data secara real-time dan bertujuan untuk mengasah keterampilan dalam menggunakan API.

Link Weather

5. Mini Project Calculator

Proyek ini adalah kalkulator berbasis web yang mendukung operasi matematika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian. Pengguna dapat memasukkan angka dan operasi melalui antarmuka pengguna yang sederhana.
